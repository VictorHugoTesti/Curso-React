# 📘 Curso-React

Repositório utilizado para acompanhar o curso **[React do Zero a Maestria (c/ hooks, router, API, Projetos)](https://www.udemy.com/course/react-do-zero-a-maestria-c-hooks-router-api-projetos/)** ministrado por **Matheus Battisti**.

---

## 🎯 Objetivo do Curso
O curso ensina **React do básico ao avançado**, cobrindo desde os recursos mais simples até os mais complexos da biblioteca mais popular do front-end.  
O foco é preparar o aluno para o mercado de trabalho, capacitando-o a criar projetos completos com integração de front-end, back-end, banco de dados e autenticação.

---

## 📚 Conteúdos Abordados
Durante o curso, foram desenvolvidos e aplicados os seguintes conhecimentos:

- Criação de projetos React via `npx` e `Vite`;
- Estrutura de pastas e arquivos em projetos React;
- Fundamentos do **JSX**;
- **Componentização** e reutilização de código;
- Estilização com **CSS**;
- Manipulação de **Formulários**;
- Consumo de **APIs** e requisições HTTP;
- Utilização da **Context API** para gerenciamento de estado;
- Implementação de **React Router** para navegação entre páginas;
- Uso de **Hooks** (useState, useEffect, useContext, entre outros);
- Exercícios práticos e projetos completos para fixação dos conceitos.

---

## 🛠️ Tecnologias Utilizadas
- **React**  
- **JavaScript ES6+**  
- **CSS**  
- **Node.js (Express)** – para criação de APIs  
- **MongoDB com Mongoose** – banco de dados NoSQL  
- **Firebase** – API, Database e Autenticação  
- **TypeScript** (em sessões bônus)
